m⁶A motif discovery in Plasmodium falciparum

Discovery of RNA motifs beyond DRACFH that are necessary for m6a modifications of mRNA.

Download PDF

At a glance

This page hosts the poster and a minimal summary so it can be followed as the project progresses. The code is not yet public.

  • Data: long-read dRNA-seq (Nanopore)
  • Features: RNA structure + gene context
  • Model: deep learning classifier for m⁶A-predictive regions
  • Motifs: enrichment on high-saliency / high-score regions
